Crear entorno virtual:

python -m venv .venv

Activar entorno virtual:

.\.venv\Scripts\Activate.ps1

Instalar dependencias:

pip install -r requirements.txt

Despues abre la paleta de comandos: Ctrl + Shift + P.

Escribe: Python: Select Interpreter y pulsa Enter.

VS Code te mostrará una lista de intérpretes.

Seleccionas el del entorno virtual, dice (recomendado) 3.12.7
